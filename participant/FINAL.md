# All Hands on Tech — Final Stage (Participants)

## Final — Assemble the access key

You should now have **four** codewords (one from each stage).

**Goal:** Combine them into the final phrase and submit it to the host.

### How to combine
- Put the codewords in order: Stage 0 → Stage 1 → Stage 2 → Stage 3
- Join them with hyphens like this:

`CODE0-CODE1-CODE2-CODE3`

✅ Submit the final phrase to the host. If correct, you win.
